# 灵曦星灿视频助手

这是一个基于火山引擎方舟平台的视频生成助手，整合了多种视频生成功能。

## 功能介绍

1. **曦光智库** - 基于 Doubao-Seed 大模型的智能文本处理与分析功能
2. **灵绘影曦** - 文生视频功能
3. **灵动画生** - 整合了多种图生视频模式：
   - 图生视频（基于首帧）
   - 图生视频（基于首尾帧）
   - 图生视频（基于参考图）

## 环境配置

1. 安装依赖：
```bash
pip install -r requirements.txt
```

2. 配置环境变量：
   - 在 `.env` 文件中配置 `ARK_API_KEY`（火山引擎方舟平台的API密钥）

## 使用方法

运行应用：
```bash
python app.py
```

应用将在 `http://0.0.0.0:7860` 上启动，可以通过浏览器访问。

## 注意事项

- 需要有效的火山引擎方舟平台API密钥
- 某些功能可能需要较长的处理时间，请耐心等待
- 现在所有图片上传功能都通过文件上传组件实现，无需手动输入URL
- 首次使用建议先尝试简单的文本提示

## 依赖库

- volcengine-python-sdk[ark] - 火山引擎方舟平台SDK
- gradio - Web界面框架
- python-dotenv - 环境变量管理
